
# Kimball’s TintWerx – Website

This zip contains everything you need to publish your site on GitHub Pages.

## Files
- `index.html` – your website
- `favicon.ico` – browser tab icon
- `social-card.png` – social preview image when your link is shared

## How to Publish on GitHub Pages
1. Go to https://github.com and create a **new repository** (e.g., `kimballs-tintwerx`).
2. Click **Add file → Upload files** and drag these three files in:
   - index.html
   - favicon.ico
   - social-card.png
3. Click **Commit changes**.
4. Go to **Settings → Pages** (left sidebar).
5. Under **Source**, select **Deploy from a branch**.
6. Choose **Branch: main** and **Folder: /(root)**, then click **Save**.
7. Wait ~1–2 minutes. Your site will be live at:
   `https://YOUR-USERNAME.github.io/kimballs-tintwerx/`

### Adding your logo & photos later
Upload these files to the same repo root when you have them:
- `kimballs-tintwerx-logo.png`
- `kimballs-tintwerx-gallery1.jpg`
- `kimballs-tintwerx-gallery2.jpg`
- `kimballs-tintwerx-gallery3.jpg`

They will automatically appear on your site (no code edits needed).
